### 2. Инструкция по развертыванию и тестированию

#### 2.1. Запуск фронтенда (React)

```bash
cd ~/ejournal-front/fr
npm install   # если зависимости еще не установлены
npm start     # запуск dev-сервера на порту 3000
```

#### 2.2. Проверка связки фронтенд ↔ бэкенд

1. Открыть браузер: `http://localhost:3000`

При подключении с удаленной машины:

```bash
ssh -L [локальный_порт]:localhost:[порт_сайта_на_сервере] [пользователь]@[ip_сервера]
```

Локальный порт: Любой свободный порт на вашем компьютере.
Порт сайта на сервере: Порт, на котором сайт запущен на сервере (3000).
Оставить этт терминал открытым.
